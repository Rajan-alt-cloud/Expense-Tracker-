# 💰 Expense Tracker Application

Welcome to the **Expense Tracker Application** — *Budget me kharch karo!*  
This is a simple Python-based console application to help you manage and track your daily expenses.

---

## 📂 Project Path


---

## 🚀 Features
- ➕ **Add Expense**: Record expenses with date, category, description, and amount.  
- 📋 **View All Expenses**: See a list of all recorded expenses.  
- 🧮 **View Total Expenses**: Calculate and display the total amount spent.  
- ❌ **Delete Expense**: Remove an expense from the record.  
- ❌ **Exit**: Quit the application gracefully.  

---

## 🛠️ Technologies Used
- 🐍 **Python 3.x**  
- 📦 **Built-in Data Structures** (Lists, Dictionaries)  
- 🎛️ **Control Flow** (Loops, Conditionals)  
- 🖥️ **Console Input/Output**
